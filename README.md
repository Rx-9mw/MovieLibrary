# How To Run
1. Pobierz i zainstaluj najnowszą wersję Microsoft .NET
2. Zainicjuj repozytorium git w folderze
```sh
git init
```
3. Skolnuj repo z linku na githubie
```sh
git clone https://github.com/Rx-9mw/MovieLibrary.git
```
4. Za pomocą terminala przejdź do ścieżki folderu z sklonowaną aplikacją
5. Odpal aplikację za pomocą komendy 
```sh
dotnet run
```
5. Po kliknięciu w wygenerowany link http://localhost:XXXX zostaniemy przekierowani do strony biblioteki filmów
